# Hugo theme Horseshoe

Horseshoe is based on [Hermit](https://github.com/Track3/hermit.git), a minimal and fast theme for Hugo.

## Features

All the features of Hermit, except:
* Nav bar at the top that hides when scrolling down
* Light theme
